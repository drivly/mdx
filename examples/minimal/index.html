MDX Minimal Example

This is a minimal example of using the mdxe package.

The application works correctly in local environments but was experiencing 404 errors when deployed to Vercel.

This static page is a temporary solution while we resolve the Vercel deployment configuration issues.

To run the full application locally:

    
    # Development mode
    pnpm dev
    
    # Production build
    pnpm build
    pnpm start
